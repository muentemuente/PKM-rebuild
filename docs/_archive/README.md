Notizen zwischen Claude-Code-Sessions und abgeschlossene Task-Specs. Sie sind kein
Teil der lebenden Projekt-Doku mehr, werden aber als Historie aufbewahrt (nicht
gelöscht — Provenance).

`docs/07_backup_strategy.md`, `README.md`) und in den Phasen-Reflexionen
